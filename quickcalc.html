Help — Calc Syntax
The calc input uses math.js expression syntax — not JavaScript. Think of it as a smart calculator with variables, units, and functions.
Variables
Assign with =. Variables persist across all lines in the same entry and can be used in later expressions.
v = 5
rate = 0.08
total = v * (1 + rate)
Basic arithmetic
Standard operators: + - * /. Use parentheses to group.
4500 * 12
(100 + 50) / 3
15 % 4   // remainder
Powers & roots
Use ^ for exponentiation (not **). Use sqrt() for square root.
5^2        // 25
2^10       // 1024
sqrt(144)  // 12
cbrt(27)   // 3
Math functions
Full math.js function library is available.
abs(-5)         // 5
round(3.7)      // 4
floor(3.9)      // 3
ceil(3.1)       // 4
log(100, 10)    // 2  (log base 10)
log(e)          // 1
sin(pi / 2)     // 1
max(3, 7, 2)    // 7
min(3, 7, 2)    // 2
Constants
pi   // 3.14159…
e    // 2.71828…
phi  // 1.61803… (golden ratio)
tau  // 2*pi
Infinity
Units
math.js supports physical units. Convert with in. Currency units (USD, EUR, etc.) are also available.
5 kg in lbs
100 km/h in mph
1 inch in cm
500 USD / 12 months
100 EUR * 1.08
What does NOT work
This is not JavaScript. The following are invalid:
var x = 6    // ✗ no JS keywords
let y = 2    // ✗
x ** 2       // ✗ use x^2 instead
console.log  // ✗ no JS APIs
if / for     // ✗ no control flow
Multiline
Press Shift+Enter for a new line in one submission. Each line is evaluated in sequence within the same scope.
