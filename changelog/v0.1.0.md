# v0.1.0 — 2026-05-14

Initial release. QuickCalc is a local-first terminal for quick math, quick thinking, and quick notes.

## Added
- Calc entries with math.js evaluation and persistent history.
- Note entries with a markdown-ready editor (CodeMirror).
- AI assistance via OpenRouter, with per-entry and global model selection.
- Tag system for organizing entries.
- Activity log capped at 500 events, with filtering by Calc, AI, Entries, Notes, and Errors.
- Settings panel covering Appearance, AI &amp; OpenRouter, Features, Data, About, and Help.
- Light and dark themes.
- Local-first persistence via `localStorage`. No account, no telemetry.
- Keyboard shortcuts: `⌘/Ctrl+K` new calc, `⌘/Ctrl+,` settings, `⌘/Ctrl+/` prompts, `Esc` close panels.
- Windows portable single-file build via `npm run build:portable`.
- About panel "View changelog" button with per-version release notes modal.
